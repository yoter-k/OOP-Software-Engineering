## Задание 01

Дана строка:
```javascript
const users = [
    "Иванов Иван Иванович",
    "Петров Пётр Сергеевич",
    "Сидорова Анна Викторовна",
    "Дмитриев Алексей Олегович",
    "Смирнов Артём Игоревич",
    "Новикова Мария Павловна",
    "Орлов Максим Андреевич",
    "Фёдорова Елена Дмитриевна"
];
```
С помощью регулярных выражений и метода `.replace()` нужно вывести:

```ruby
Иван Иванов
Пётр Петров
Анна Сидорова
Дмитриев Алексей
Артём Смирнов
Мария Новикова
Максим Орлов
Елена Фёдорова
```

Реализовано двумя способами:
- `task1_a.js` — нумерованные группы
- `task1_b.js` — именованные группы

## Задание 02

Дана строка с объектом:
```javascript
let json = `
{ 
    "a": 1, 
    "b": { "c": 2, "d": 3 }, 
    "e": 4, 
    "fff": { "v": 10 } 
};`;
```
С помощью регулярных выражений получены:
```python
1) Только значения-объекты  
[ '{ "c": 2, "d": 3 }', '{ "v": 10 }' ]

2) Только имена полей  
[ 'b', 'fff' ]

3) Пары имя–значение  
[ [ 'b', '{ "c": 2, "d": 3 }' ], [ 'fff', '{ "v": 10 }' ] ]
```
Файлы:
- `task2_a.js`
- `task2_b.js`
- `task2_c.js`
