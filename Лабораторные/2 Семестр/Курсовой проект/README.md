# Курсовой проект: Веб-приложение «Игровой Атлас»

Интерактивный веб-справочник лучших видеоигр, разработанный в рамках курсового проекта по программной инженерии. Приложение построено по классической архитектуре **MVC** (Model-View-Controller) на платформе **Node.js** с использованием веб-фреймворка **Express** и шаблонизатора **EJS**.

---

## 🌟 Ключевые возможности приложения

* **Каталог видеоигр**: Полный интерактивный список лучших игровых релизов с детальной информацией (студия-разработчик, жанр, год выпуска, описание и оценки).
* **Разграничение ролей (ACL)**: 
  * `Пользователь` — просмотр каталога, интерактивный поиск, фильтрация, управление списком «Избранного» в личном кабинете.
  * `Администратор` — имеет доступ ко всем функциям пользователя, а также к панели добавления новых игр и к разделу просмотра зарегистрированных пользователей.
* **Интерактивный SPA-поиск**: Умный мгновенный поиск по названию, жанру или разработчику без перезагрузки страницы.
* **Фильтрация и сортировка**: Удобные фильтры по жанрам, а также сортировка по рейтингу критиков, оценкам игроков, году релиза и названию.
* **Система «Избранного»**: Возможность добавлять игры в закладки в реальном времени (AJAX/Fetch API) с отображением в личном кабинете.
* **Экспорт данных в Excel**: Генерация стилизованного отчета по текущей выборке игр с защитой от автоматического форматирования чисел в даты (баг Excel с русским языком).
