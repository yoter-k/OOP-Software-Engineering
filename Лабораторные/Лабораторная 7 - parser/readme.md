# NBA Parser

Парсит турнирные таблицы НБА 2025 с сайта [Basketball Reference](https://www.basketball-reference.com/leagues/NBA_2025_standings.html) и сохраняет данные в CSV и JSON форматах.

![alt text](image.png)

---

## Описание работы

- Загружает HTML страницы и парсит таблицы конференций (Eastern/Western) с помощью `cheerio`.
- Сохраняет данные о командах: победы, поражения, процент побед, очки за/против, SRS.
- CSV-файл содержит подписи конференций.
- JSON-файл содержит структурированные объекты команд с той же информацией.
